AM Modulation Hardware & Circuitry

SQUARE-LAW MODULATOR

Non-Linear Direct Addition

Utilizes non-linear operation region (square-law portion) of a diode or FET. Message $m(t)$ and Carrier $c(t)$ pass summation $v_1 = m+c$, then expand by $v_2 = a v_1 + b v_1^2$.

Input Waveform: Continuous superposition of slow baseband on fast RF.
Output Theorem: $s(t)$ is filtered from $2bm(t)c(t)$ term by Tank BPF.
SWITCHING MODULATOR

High-Speed Carrier Gating

Drives a diode bridge purely as a periodic ON/OFF switch via dominant carrier voltage magnitude. Effectively multiplies input message by square periodic wave $p(t)$.

Input Waveform: Pure analog message gated into pulses.
Output Theorem: Fundamental harmonic extraction via resonant BPF outputs perfect AM sinusoid.
COLLECTOR MODULATOR

High-Level (Class C) Driver

BJT Transistor configured in Class C. The base is driven into saturation by the carrier, while the collector DC voltage is dynamically modulated directly by the audio message via step-up transformer.

Input Waveform: Pulsed base current with shifting collector rail.
Output Theorem: The high-power tank flywheel converts rail-shifting pulses into maximal efficiency continuous AM wave.
